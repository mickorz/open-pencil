# OpenPencil 国际化（i18n）设计文档

## 概述

为 OpenPencil UI 添加多语言支持，以中文为默认语言，支持中英文切换。

---

## 需求

| 项目 | 选择 |
|-----|------|
| 语言范围 | 中文 + 英文 |
| 默认语言 | 中文 |
| 切换方式 | 界面手动切换 |
| 持久化 | localStorage |
| i18n 库 | vue-i18n |
| 方案 | 最小化集成 |

---

## i18n 配置

```typescript
// src/i18n.ts
import { createI18n } from 'vue-i18n'
import zhCN from './locales/zh-CN'
import enUS from './locales/en-US'

const STORAGE_KEY = 'open-pencil:locale'

function getSavedLocale(): string {
  return localStorage.getItem(STORAGE_KEY) || 'zh-CN'
}

const i18n = createI18n({
  legacy: false,
  locale: getSavedLocale(),
  fallbackLocale: 'zh-CN',
  messages: {
    'zh-CN': zhCN,
    'en-US': enUS
  }
})

export function setLocale(locale: string): void {
  i18n.global.locale.value = locale
  localStorage.setItem(STORAGE_KEY, locale)
}

export function getCurrentLocale(): string {
  return i18n.global.locale.value
}

export default i18n
```

## main.ts 集成

```typescript
// src/main.ts
import { createApp } from 'vue'
import App from './App.vue'
import router from './router'

const app = createApp(App)
app.use(router)
app.use(i18n)
app.mount('#app')
```

## 语言切换 UI

**位置**: AppMenu.vue 右侧

**组件**:
```vue
<template>
  <DropdownMenuRoot>
    <DropdownMenuTrigger class="locale-trigger">
      {{ currentLocaleLabel }}
    </DropdownMenuTrigger>
    <DropdownMenuContent>
      <DropdownMenuItem @select="setLocale('zh-CN')">
        中文
      </DropdownMenuMenuItem>
      <DropdownMenuItem @select="setLocale('en-US')">
        English
      </DropdownMenuItem>
    </DropdownMenuContent>
  </DropdownMenuRoot>
</template>

<script setup lang="ts">
import { computed } from 'vue'
import { useI18n } from 'vue-i18n'
import { setLocale } from '@/i18n'
import {
  DropdownMenuRoot,
  DropdownMenuTrigger,
  DropdownMenuContent,
  DropdownMenuItem
} from 'reka-ui'

const { locale } = useI18n()

const currentLocaleLabel = computed(() => {
  return locale.value === 'zh-CN' ? '中文' : 'EN'
})
</script>
```

## 组件使用示例

```vue
<script setup lang="ts">
import { useI18n } from 'vue-i18n'
const { t } = useI18n()
</script>

<template>
  <button>{{ t('common.save') }}</button>
  <span>{{ t('toolbar.select') }}</span>
</template>
```

## 实施步骤

1. 安装 vue-i18n 依赖
2. 创建语言包文件 (zh-CN.ts, en-US.ts)
3. 创建 i18n 配置 (i18n.ts)
4. 在 main.ts 中集成 i18n
5. 改造各组件使用 i18n
6. 添加语言切换 UI
7. 测试验证

## 注意事项

1. 使用 Composition API 模式 (`legacy: false`)
2. 所有文本必须通过 `t()` 函数获取
3. 新增功能需同步添加翻译
4. localStorage key: `open-pencil:locale`
